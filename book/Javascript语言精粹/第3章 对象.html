# 第3章 对象

  1. 对象属性名可以包括空字符串的任意字符串
  var obj={
    undefined:undefined,
    '':1,
    ' ':2,
    's-s':3
    //s-s:2//报错，属性名中不能有连接符（-）
  }
  console.log(obj['undefined'])//undefined
  console.log(obj.undefined)//undefined
  console.log(obj['s-s'])//3
  console.log(obj[' '])//2
  console.log(obj[''])//1
